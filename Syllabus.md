
**Scope**

This course introduces computer applications, Python programming, healthcare

information systems, web technologies, database systems and introductory Artificial

Intelligence concepts relevant to pharmaceutical sciences. The course enables students to

manage, analyze and visualize healthcare and pharmaceutical datasets while developing

computational skills required for modern pharmacy practice.

**Course Objectives**

• Introduce fundamental computer applications used in pharmaceutical sciences.

• Develop programming skills using Python for scientific and healthcare applications.

• Familiarize students with web technologies and database systems.

• Enable handling, analysis and visualization of pharmaceutical datasets.

• Introduce digital healthcare information systems, pharmacy databases and basic AI

applications.

• Prepare students for advanced AI, Machine Learning and Healthcare Analytics courses.

**Course Outcomes**

• CO1: Explain computer fundamentals, web technologies, Python programming and basic

AI concepts.

• CO2: Develop Python programs using control structures, functions and data structures.

• CO3: Design and manage healthcare databases and retrieve pharmaceutical information.

• CO4: Analyze pharmaceutical datasets using NumPy and Pandas.

• CO5: Visualize healthcare data and understand basic AI applications in pharmaceutical

sciences.

**UNIT I – Artificial Intelligence, Computer Applications & Python Programming in**

**Healthcare**

Introduction to Artificial Intelligence (AI): Concepts of AI, AI applications in healthcare and

pharmacy, AI-assisted drug discovery, clinical decision support systems, and digital

healthcare innovations. Introduction to Computer Applications in Pharmacy: Role of

Information Technology in Pharmaceutical Sciences, Digital Healthcare Systems, Electronic

Health Records (EHR), and Pharmacy Information Systems. Python installation and

Integrated Development Environments (Jupyter Notebook and VS Code), variables, data

types, operators, input/output operations, string manipulation techniques, and introduction

to standard libraries.

**UNIT II – Programming Logic, Functions and Healthcare Applications**

Programming Fundamentals using Python: Conditional statements, looping constructs,

user-defined functions, passing arguments and returning values. Development of simple

healthcare and pharmaceutical applications such as dosage calculation, BMI calculation, unit

conversion systems, prescription-related computations and introduction to AI-assisted

problem-solving concepts.

**UNIT III – Data Structures, File Handling and Healthcare Databases**

Data Management in Pharmaceutical Sciences: Lists, tuples and dictionaries, indexing and

slicing operations, introduction to NumPy arrays and basic numerical computations.

Reading and writing CSV files, structured healthcare datasets, introduction to database

concepts including tables, records, fields and keys, healthcare databases, pharmacy drug

databases, patient information systems and retrieval of pharmaceutical information.

**UNIT IV – Web Technologies, Database Systems and Data Handling**

Web Technologies and Healthcare Information Systems: Fundamentals of HTML, overview

of web applications and role of programming languages in web-based healthcare systems.

Introduction to MySQL and MS Access, healthcare database management, drug information

systems and clinical data repositories. Pandas Series and DataFrames, reading CSV and

Excel files, data cleaning, handling missing values and filtering of healthcare and

pharmaceutical datasets.

**UNIT V – Data Analysis, Visualization and AI Applications in Pharmacy**

Data Analysis and Decision Support Systems: NumPy operations for scientific computing,

data visualization using Matplotlib including line plots and scatter plots. Visualization and

interpretation of pharmaceutical datasets such as concentration-time profiles, dissolution

studies and adverse drug reaction reports. Overview of Artificial Intelligence and Machine

Learning applications in drug discovery, healthcare analytics, clinical decision support

systems and introduction to mini-project development using healthcare and pharmaceutical

datasets

5. Create a database in MS Access to store patient information with required fields.

6. Design a form in MS Access to view, add, delete and modify patient records.

7. Generate and print reports from the patient database.

8. Create an invoice table using MS Access.

9. Drug information storage and retrieval using MS Access.

10. Creating and working with queries in MS Access.

11. Exporting Tables, Queries, Forms and Reports to web pages.

12. Exporting Tables, Queries, Forms and Reports to XML pages.List of Experiments AI & Python Programming

13. Develop a Pharmaceutical Calculator using Python.

14. Analyze Healthcare Data using Python and Pandas.

15. Visualize Pharmaceutical Data using Python and Matplotlib.
